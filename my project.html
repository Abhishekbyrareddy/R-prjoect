Introduction

About my Dataset

Dataset Description

The dataset comprises economic indicators data, merged from two separate Excel files: “DS project3.xls” and “DS project4.xls”. These datasets contain time-series data with observations on key economic metrics over a specified period.

Data Sources

  • DS project3.xls: This file includes data on the Consumer Price Index for All Urban Consumers: All Items Less Food and Energy (CPILFESL).

  • DS project4.xls: This file includes data on the Effective Federal Funds Rate (FEDFUNDS).

Variables

  1. observation_date: The date of the observation, typically formatted as YYYY-MM-DD.

  2. CPILFESL: Consumer Price Index for All Urban Consumers: All Items Less Food and Energy. This index measures the changes in the price level of a market basket of consumer goods and services, excluding food and energy prices, which tend to be more volatile.

  3. FEDFUNDS: Effective Federal Funds Rate. This is the interest rate at which depository institutions trade federal funds (balances held at Federal Reserve Banks) with each other overnight. It is a key indicator of monetary policy.

Merging Process

The two datasets are merged based on the observation_date column, ensuring that each observation includes corresponding values for both CPILFESL and FEDFUNDS.

Purpose

The merged dataset is used to analyze the relationships between the consumer price index and the federal funds rate over time. This analysis helps to understand the impact of monetary policy (represented by the federal funds rate) on inflation and economic stability (represented by the consumer price index).

Visualization

Two line plots are created to visualize the data:

  1. Response to Fiscal Shock: Shows the trends in CPILFESL and FEDFUNDS over time.

  2. Response to Interest Rate Shock: Shows the trends in CPILFESL and FEDFUNDS over time, highlighting different economic conditions.

These visualizations help in understanding how changes in interest rates (FEDFUNDS) correlate with changes in consumer prices (CPILFESL), providing insights into the effectiveness of fiscal policies.

Usage

Researchers, economists, and policymakers can use this dataset to:

  • Study the historical relationship between inflation (excluding food and energy prices) and interest rates.

  • Analyze the effects of monetary policy changes on the economy.

  • Make informed decisions based on historical data trends.

What is R and R markdown

R is a programming language and software environment designed for statistical analysis, data manipulation, and graphical representation. It is popular among statisticians, data scientists, and researchers due to its robust capabilities in handling and visualizing data. R is free, open-source, and can be extended with numerous packages available on the Comprehensive R Archive Network (CRAN).

R Markdown is a versatile document format that allows you to embed R code within narrative text, producing documents in formats like HTML, PDF, and Word. It facilitates reproducible research by combining data analysis and reporting into a single, cohesive document, enabling the creation of dynamic reports that seamlessly integrate both code and results.

What is Quorto

Quarto is an open-source publishing system for creating dynamic documents, reports, presentations, and websites. It supports multiple languages like R, Python, Julia, and JavaScript, allowing integration of code, text, and output. Quarto offers flexible output formats, enhanced collaboration, and extensive customization, facilitating reproducible research and technical documentation.
